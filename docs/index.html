Five Integrated Innovations

Innovation 1 — §2.3

Dual-Space Khatri-Rao Reformulation

Transfers trace estimation to a symmetric PSD Gram matrix of size $(m_A m_B)^2$, enabling standard Hutch++ with quadratic variance reduction. Theorem 1 guarantees eigenvalue equivalence.

Innovation 2 — §2.2

Generalized FWL Orthogonalization

Proposition 1: exact main-effect annihilation for both continuous and binary traits. IRLS weights embedded in V — no explicit weight injection needed (Remark 1).

Innovation 3 — §2.4

Non-Linear Genomic Control

Eliminates phantom epistasis using sparse Hadamard-squared GRM $\boldsymbol{\Phi} \odot \boldsymbol{\Phi}$ and randomized Haseman-Elston regression for two-component variance estimation.

Innovation 4 — §2.5

CKKS Homomorphic Encryption

Fed-cSPA-HE protocol: only 5 encrypted scalars leave each node. Homomorphic addition aggregates ciphertexts without decryption — zero raw genotype leakage.

Innovation 5 — §2.6

Graph-Regularized Search Space

Union of AlphaFold-Multimer PPI (iPTM ≥ 0.5), TAD co-localization from Micro-C, and KEGG/Reactome pathway co-membership. Reduces ~2×10⁸ hypotheses to ~10⁴ candidates.

Mathematical Foundation

Base Model & Null Projection §2.1

$$g(\boldsymbol{\mu}) = \mathbf{X}\boldsymbol{\alpha} + \mathbf{u}, \quad \mathbf{u} \sim \mathcal{N}(\mathbf{0}, \tau^2 \boldsymbol{\Phi})$$
$$\mathbf{V} = \mathbf{W}^{-1} + \tau^2 \boldsymbol{\Phi}$$
$$\mathbf{P}_0 = \mathbf{V}^{-1} - \mathbf{V}^{-1}\mathbf{X}(\mathbf{X}^\top \mathbf{V}^{-1}\mathbf{X})^{-1}\mathbf{X}^\top \mathbf{V}^{-1}$$
Remark 1: IRLS weights are already embedded in V. No explicit weight injection needed for binary traits.

Generalized FWL Proposition 1

$$\mathbf{P}_{\mathrm{adj}} = \mathbf{P}_0 - \mathbf{P}_0 \mathbf{Z}_{\mathrm{main}} (\mathbf{Z}_{\mathrm{main}}^\top \mathbf{P}_0 \mathbf{Z}_{\mathrm{main}})^{-1} \mathbf{Z}_{\mathrm{main}}^\top \mathbf{P}_0$$
Symmetric, V-metric idempotent ($\mathbf{P}_{\mathrm{adj}}\mathbf{V}\mathbf{P}_{\mathrm{adj}} = \mathbf{P}_{\mathrm{adj}}$), and $\mathbf{P}_{\mathrm{adj}}\mathbf{Z}_{\mathrm{main}} = \mathbf{0}$ exactly.

Dimensionality Collapse Proposition 2

$$Q_{\mathrm{adj}} = \tfrac{1}{2} \left\lVert \mathbf{Z}_A^\top \mathrm{Diag}(\tilde{\mathbf{y}}) \mathbf{Z}_B \right\rVert_F^2 \equiv \tfrac{1}{2} \tilde{\mathbf{y}}^\top \mathbf{K}_{\mathrm{epi}} \tilde{\mathbf{y}}$$
Complexity: $O(N \cdot m_A \cdot m_B)$ — no N×N matrix ever formed.

Dual-Space Reduction Theorem 1

$$\mathbf{G}_{\mathrm{dual}} = \mathbf{Z}_{\mathrm{KR}}^\top \mathbf{P}_{\mathrm{adj}} \mathbf{Z}_{\mathrm{KR}}$$
$(m_A m_B) \times (m_A m_B)$ symmetric PSD matrix. Non-zero eigenvalues identical to the N×N $\mathbf{P}_{\mathrm{adj}}\mathbf{K}_{\mathrm{epi}}$.

Hutch++ Deflation Algorithm 1

$$\mathrm{Var}[\hat{t}_p] \leq \frac{2}{S} \left\lVert \mathbf{G}_{\mathrm{dual}} - \mathbf{G}_k \right\rVert_F^2$$
Nyström deflation of top-k eigenspace. Residual Hutchinson on well-conditioned remainder. Quadratic variance improvement.

Lugannani-Rice SPA §2.5

$$P(Q > q) \approx \bar{\Phi}(\hat{w}) + \phi(\hat{w})\left(\frac{1}{\hat{w}} - \frac{1}{\hat{u}}\right)$$
$$\hat{w} = \mathrm{sign}(\hat{t})\sqrt{2(\hat{t}q - K(\hat{t}))}, \quad \hat{u} = \hat{t}\sqrt{K''(\hat{t})}$$
Computed via erfc() for exact precision to $P \approx 10^{-300}$.

Non-Linear Genomic Control §2.4

$$\mathbf{V}_{\mathrm{aug}} = \sigma_e^2 \mathbf{I} + \tau^2 \boldsymbol{\Phi} + \tau_{\mathrm{epi}}^2 (\boldsymbol{\Phi} \odot \boldsymbol{\Phi})$$
Sparse GRM sparsification + randomized Haseman-Elston regression. Eliminates phantom epistasis in inbred populations.

Cumulant Additivity Theorem 2

$$\kappa_{j,\mathrm{meta}} = \sum_{k=1}^{K} \kappa_{j,k} \quad \forall\, j \geq 1$$
Only 5 encrypted scalars per node. Fed-cSPA-HE protocol with CKKS homomorphic encryption — zero genotype leakage.
