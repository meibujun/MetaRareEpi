System Architecture

🧮

Kernel Engine

kernel/fast_mvm.py

Implicit epistatic kernel MVM: $K_{epi} @ v$ without forming the dense $N \times N$ matrix. Two-step einsum: $\mathcal{O}(N \cdot m_A \cdot m_B)$.

📐

SPA Engine

spa/saddlepoint.py

Halley's method saddlepoint solver + Lugannani-Rice formula. Uses norm.sf() for tail precision to $P \approx 10^{-300}$.

🌐

Federated Layer

federated/node.py

Ray-based remote actors. Each node computes local $\hat{\kappa}_{1..4}$ and transmits only 4 scalars. Zero genotype leakage.

📊

GLMM Base

glmm.py

Null model engine: AI-REML variance components, $P_0$ projection, and whitened residuals $\tilde{y} = P_0(y - \hat{\mu})$.

🎯

Variant Weighting

weighting.py

Pluggable annotation scoring: CADD, AlphaMissense, Beta(MAF). Produces $Z = G \cdot W$ weighted feature matrices.

💾

Zarr I/O

io/zarr_store.py

Out-of-core Zarr v3 handler for chunked genotype access. Context manager with column-standardisation support.

Mathematical Foundations

FWL-Orthogonalized Score Statistic

$$Q_{adj} = \frac{1}{2} \left\| \mathbf{Z}_A^T \text{Diag}(\mathbf{y}_{adj}^*) \mathbf{Z}_B \right\|_F^2$$

The epistatic variance component score under FWL-orthogonalization collapses to a squared Frobenius norm of an $m_A \times m_B$ matrix, bypassing the $\mathcal{O}(N^2)$ covariance tensor entirely. This guarantees immunity to main-effect confounding.

Implicit Fast-MVM Identity

$$\text{Tr}((\mathbf{P}_{adj}\mathbf{K}_{epi})^j) \approx \frac{1}{S} \sum_{s=1}^S \mathbf{r}_s^T (\mathbf{P}_{adj}\mathbf{K}_{epi})^j \mathbf{r}_s$$

Higher-order cumulant traces are estimated via Hutchinson's stochastic estimator with $S$ Rademacher vectors, coupled with the implicit MVM iteration: $\mathbf{w}^{(i)} = [(\mathbf{Z}_A \mathbf{C}^{(i)}) \odot \mathbf{Z}_B] \mathbf{1}_{m_B}$ where $\mathbf{C}^{(i)} = \mathbf{Z}_A^T \text{Diag}(\mathbf{v}) \mathbf{Z}_B$. Each iteration is $\mathcal{O}(N \cdot m_A \cdot m_B)$.

Cumulant Additivity Theorem

$$\kappa_{j, meta} = \sum_{k=1}^K \kappa_{j, k} \quad \forall j \ge 1$$

For independent cohorts, the global CGF is the sum of local CGFs. This enables lossless federated meta-analysis by transmitting only 4 scalar cumulants per node — zero raw genotype data leakage while preserving 100% analytical equivalence to centralized mega-analysis.

Lugannani-Rice Tail Probability

$$P(X > q) \approx \bar{\Phi}(\hat{w}) + \phi(\hat{w})\left(\frac{1}{\hat{w}} - \frac{1}{\hat{u}}\right)$$

Where $\hat{w} = \text{sign}(\hat{t})\sqrt{2(\hat{t}q - K(\hat{t}))}$ and $\hat{u} = \hat{t}\sqrt{K''(\hat{t})}$. The survival function $\bar{\Phi}$ is computed via erfc() for exact precision to $P \approx 10^{-300}$.

pip install -e ".[dev]"

python -m pytest tests/ -v --tb=short

🚀 Quick Example

import numpy as np
from metararepi.spa import spa_pvalue
from engine_jax import extract_local_cumulants

Z_A = np.random.randn(5000, 20)
Z_B = np.random.randn(5000, 20)

result = extract_local_cumulants(Z_A, Z_B, method="exact")
pval = spa_pvalue(15.0, result["cumulants"])
print(f"P = {pval['pvalue']:.2e}")
